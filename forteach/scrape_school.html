任務與代碼提交

同學你好，請使用 soup.select() 解析右側的「濠江中學模擬網站」。請嚴格按照題目的限制符號作答，並將核心 Python 代碼貼在每個任務下方，完成後點擊最下方按鈕送出！

任務 01 [父子關係]：必須使用 > 符號。

目標：獲取「最新公告區」中，直接屬於 .board 的 .notice 文字。請避開被包在深層的錯誤公告。

提示：soup.select(".board > .notice") 確保只找下一層的子節點。

任務 02 [祖先後代]：必須使用 (空格) 符號。

目標：獲取「資工系課程表」中，ID 為 science-class 底下所有 class 為 course 的課程名稱，不管包了幾層都要抓到。

提示：soup.select("#science-class .course") 利用空格能穿透所有階層尋找。

任務 03 [屬性定位]：必須使用 [ ] 符號。

目標：在圖書館清單中，找出所有自訂屬性 data-borrow="yes" (可借閱) 的書名。

提示：soup.select("[data-borrow='yes']") 能直接鎖定具備特定屬性值的標籤。

任務 04 [唯一識別]：必須使用 # 符號。

目標：在校長信箱區塊，有一組隱藏的校園免費 WIFI 密碼，它的 ID 是 hidden-wifi，請提取它的文字。

提示：soup.select("#hidden-wifi") ID 是網頁中的唯一值，加上 # 就能瞬間找到。

任務 05 [模糊匹配]：必須使用 *= 符號。

目標：在社團招募連結中，找出所有 href 網址中「包含」 club_reg 字眼的網址。

提示：soup.select("a[href*='club_reg']") 可以抓出部分相符的屬性內容！

任務 06 [標籤與類別結合]：必須同時指定標籤與 class。

目標：在「榮譽榜」中，獲取所有 li 標籤且 class 為 honor-student 的學生姓名。

提示：寫法為 soup.select("標籤名.class名")，中間不要有空格喔！

任務 07 [ID與後代結合]：結合 # 與空格。

目標：獲取 ID 為 lunch-menu 的區塊中，所有 class 為 dish 的菜色名稱。

提示：這題是把任務2跟任務4結合，試試 soup.select("#ID名稱 .class名稱")。

任務 08 [列表索引]：Python 列表操作。

目標：獲取 class 為 club-name 的社團列表，並用 Python 提取「第 4 個」社團的名字。

提示：抓出列表後，記得使用 [3].text 來提取第 4 個元素 (因為索引從 0 開始算)。

任務 09 [字串過濾]：結合 Python 的 in 關鍵字。

目標：獲取所有 class 為 exam-rule 的段落，統計裡面包含 "禁止" 兩個字的段落數量。

提示：先選出所有規則，再使用 for 迴圈搭配 if "禁止" in x.text: 來進行計數。

任務 10 [屬性與後代結合]：結合 [ ] 與空格。

目標：獲取屬性 data-grade="senior" (高中部) 區塊底下，class 為 class-title 的班級名稱。

提示：結合任務3跟任務2，寫法如 soup.select("[屬性='值'] .class名稱")。

老師後端設定 (Google Sheet API)

此處設定會直接保存在這台電腦的瀏覽器 (LocalStorage) 中，學生開啟網頁時預設不會看到。請在此貼上您的 Google Apps Script Web App 網址以接收學生的代碼。

※ 已經為您預設綁定好了，如非必要無需更改！

5分鐘建置 Google 試算表後端教學：

  1. 建立一個新的 Google 試算表。
  2. 點選上方選單的 「擴充功能」 -> 「Apps Script」。
  3. 將裡面的程式碼清空，並完整貼上下方的 GAS 範例代碼。
  4. 點選右上角的 「部署」 -> 「新增部署」。
  5. 選取類型為 「網頁應用程式」：
    • 專案負責人：「您自己」
    • 誰有權限存取：選擇 「任何人 (Anyone)」 (關鍵設定！)
  6. 點擊部署，並授權存取權限。複製產生的 「網頁應用程式網址」 並貼到上方欄位即可！

function doPost(e) {
  try {
    var params = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(["時間", "組別", "任務1", "任務2", "任務3", "任務4", "任務5", "任務6", "任務7", "任務8", "任務9", "任務10"]);
      sheet.setFrozenRows(1);
    }
    
    sheet.appendRow([
      new Date(), params.leaderName,
      params.code01, params.code02, params.code03, params.code04, params.code05,
      params.code06, params.code07, params.code08, params.code09, params.code10
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "success", "message": "提交成功！" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
