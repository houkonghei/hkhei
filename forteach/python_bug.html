任務與代碼提交

特工，請使用 soup.select() 解析右側模擬資料。請將你們成功獲取目標的核心 Python 代碼貼在每個任務下方的輸入框中，完成後點擊最下方的「提交小組代碼」按鈕送出！

任務 01：獲取本網頁的 <title> 文字。

提示：使用 soup.select("title") 會回傳一個列表，記得用 [0] 取出第一個元素後再加上 .text！

任務 02：找到 ID 為 secret-message 的隱藏段落，並提取其文字。

提示：在 CSS 選擇器中，ID 前面要加井字號，試試 soup.select("#secret-message")。

任務 03：獲取所有商品名稱 (class 為 product-name) 並存成列表。

提示：class 前面要加小數點。使用 soup.select(".product-name") 抓出列表後，可以用 for 迴圈將 .text 裝進 list。

任務 04：計算表格中所有「銷售額」(class 為 sales) 的數字總和。

提示：用 soup.select(".sales") 抓取後，記得將每個項目的 .text 透過 int() 轉換成整數才能相加。

任務 05：找到聯絡管理員區塊，提取其自訂屬性 data-email 的信箱地址。

提示：可以用 soup.select("div#contact-info") 找到標籤，然後像查字典一樣用 [0]["data-email"] 取出屬性。

任務 06：找到 class 為 target-link 的標籤，並提取其 href 網址。

提示：與任務 5 觀念相同，先 soup.select(".target-link") 取出第 0 個元素，再用 ["href"] 拿取網址。

任務 07：在評論區中，獲取「第五個」評論者的名字 (class 為 author)。

提示：soup.select(".author") 產生的列表索引是從 0 開始算起的，想想第五個元素的索引是 [?]。

任務 08：找出被標記為售完 (class 包含 sold-out) 的商品，並提取其價格。

提示：可以利用屬性選擇器的模糊比對，例如 soup.select("[class*=sold-out]") 來精準鎖定標籤。

任務 09：獲取屬性為 rel="next" 的「下一頁」連結網址。

提示：select 完全支援中括號屬性選擇器，直接試試 soup.select('a[rel="next"]')。

任務 10：統計 class 為 article 的段落中，包含 "Python" 關鍵字的文章數量。

提示：用 soup.select('p.article') 抓出所有文章，用 for 迴圈搭配 if "Python" in x.text: 來計數。

5分鐘建置 Google 試算表後端教學：

  1. 建立一個新的 Google 試算表。
  2. 點選上方選單的 「擴充功能」 -> 「Apps Script」。
  3. 將裡面的程式碼清空，並完整貼上下方的 GAS 範例代碼。
  4. 點選右上角的 「部署」 -> 「新增部署」。
  5. 選取類型為 「網頁應用程式」：
    • 專案負責人：「您自己」
    • 誰有權限存取：選擇 「任何人 (Anyone)」 (關鍵設定！)
  6. 點擊部署，並授權存取權限。複製產生的 「網頁應用程式網址」 並貼到上方欄位即可！

function doPost(e) {

    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // 寫入資料：時間戳記、組長姓名、任務1 ~ 任務10 的代碼
    sheet.appendRow([
      new Date(),
      params.leaderName,
      params.code01,
      params.code02,
      params.code03,
      params.code04,
      params.code05,
      params.code06,
      params.code07,
      params.code08,
      params.code09,
      params.code10
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "success", "message": "提交成功！" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
