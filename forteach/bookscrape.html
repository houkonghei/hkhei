任務 01：提取 We love being scraped 的口號。

目標：獲取網站頂部導航條或標題區下方的口號文字："We love being scraped!"。

提示：口號藏在特定容器下的 small 標籤，試試：soup.select("div > small")。

任務 02：提取本頁書籍名稱並給予編號。

目標：獲取右側展示的所有書本名稱，在 Python 端使用迴圈印出編號與書名（注意：網頁展示的文字較長時可能會被省略成 ...）。

提示：每本書都被包在 h3 裡面的 a 標籤中，試試用 soup.select('h3 a')，再以 for i in range(...) 跑迴圈印出編號與 .text。

任務 03：提取完整書名（隱藏在 HTML 屬性中）。

目標：當書名太長時，.text 會顯示省略號。請提取隱藏在 <a> 標籤內部的完整書名（即 title 屬性的內容）。

提示：可以使用 soup.select('h3 a') 找到所有連結物件，並使用 Python 字典取值語法 x["title"] 提取完整內容。

任務 04：提取各個書本價格，以及價格的總和。

目標：提取網頁中所有書籍價格，刨除英鎊符號 `£`，將價格轉換為浮點數（float）並計算這一頁書籍的總價格。

提示：書籍價格標籤的 class 是 price_color。提取後使用切片 bp.text[1:] 跳過第一個貨幣符號，再用 float() 轉換累加。

任務 05：計算評分為「二星 (Two)」的書本項目共有多少本。

目標：Books to Scrape 網頁使用 class 表示星級評分（如 One, Two, Three...）。請統計本頁面共有幾本書得到了「二星」評分。

提示：二星書籍段落同時有 star-rating 與 Two 這兩個 class。在 CSS 選擇器中，多重 class 連寫如 p.star-rating.Two，再用 len() 計算列表長度。

任務 06：提取左側的書籍分類欄目。

目標：獲取側邊欄中所有的書籍分類名稱。注意：獲取文字時記得剔除多餘的空白換行字元。

提示：可以使用父子關係選擇器 li > a 來定位分類選單的超連結，在 Python 中使用 .text.strip() 函數來完美清除多餘空格。

老師後端設定 (Google Sheet API)

此處設定會直接保存在這台電腦的瀏覽器 (LocalStorage) 中，學生開啟網頁時預設不會看到。請在此貼上您的 Google Apps Script Web App 網址以接收學生的代碼。

※ 已經為您預設綁定好了，如非必要無需更改！

5分鐘建置 Google 試算表後端教學：

  1. 建立一個新的 Google 試算表。
  2. 點選上方選單的 「擴充功能」 -> 「Apps Script」。
  3. 將裡面的程式碼清空，並完整貼上下方的 GAS 範例代碼。
  4. 點選右上角的 「部署」 -> 「新增部署」。
  5. 選取類型為 「網頁應用程式」：
    • 專案負責人：「您自己」
    • 誰有權限存取：選擇 「任何人 (Anyone)」 (關鍵設定！)
  6. 點擊部署，並授權存取權限。複製產生的 「網頁應用程式網址」 並貼到上方欄位即可！

📋 GAS 範例代碼 (為 6 題重構，可直接複製)：

function doPost(e) {
  try {
    var params = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(["時間", "組別", "任務1 (口號)", "任務2 (書名與編號)", "任務3 (完整書名)", "任務4 (價格與總和)", "任務5 (二星書數)", "任務6 (分類選單)"]);
      sheet.setFrozenRows(1);
    }
    
    sheet.appendRow([
      new Date(), params.leaderName,
      params.code01, params.code02, params.code03, params.code04, params.code05, params.code06
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "success", "message": "提交成功！" }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
