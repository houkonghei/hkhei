藍色階梯：目前正在計算的目標點。

在 DP 中，我們不是在算「怎麼走」，而是在算「有多少種走法」。

要到達第 i 階，只有兩條路：
1. 從 i-1 跨一步
2. 從 i-2 跨兩步

2 C++ 實作程式碼 

// 爬樓梯問題的 DP 解法
int climbStairs(int n) {
    if (n <= 2) return n;
    
    // 1. 定義狀態陣列
    int dp[n + 1];
    
    // 2. 設定初始邊界
    dp[1] = 1; // 1階只有1種走法
    dp[2] = 2; // 2階有2種走法(1+1, 2)
    
    // 3. 狀態轉移 (遞推填表)
    for (int i = 3; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    
    return dp[n];
}

初學者常犯錯誤

忘記初始化 `dp[1]` 和 `dp[2]`。如果沒有這些基礎值，後面的計算就像沒有地基的房子，會算不出結果。

為什麼叫「動態規劃」？

「動態」指的是你的決定會影響後續，「規劃」在數學中其實是指「查表」或「最優化」的方法。
