八皇后问题

C++ 回溯算法编程教学与互动演示

什么是八皇后问题？

八皇后问题是一个古老而著名的问题，由国际象棋棋手马克斯·贝瑟尔于1848年提出。 它是回溯算法的经典教学案例，被广泛用于数据结构与算法的入门教学。

问题规则

  • 在一个 8×8 的国际象棋棋盘上放置 8 个皇后
  • 任意两个皇后都不能互相攻击
  • 皇后可以攻击同一行、同一列、同一对角线上的任意棋子
  • 也就是说：任意两个皇后都不能处于同一行、同一列或同一斜线上

问题的解

八皇后问题共有 92 种不同的独立解。 如果我们将通过旋转、翻转棋盘得到的相同解视为同一个解的话，那么最终只有 12 个独立的基本解。

💡 暴力枚举需要检查 44 亿种可能的摆放方式，而通过回溯算法，我们可以高效地剪枝掉大量无效的搜索空间，快速找到所有解。

回溯算法思路

回溯法（Backtracking）是一种暴力搜索算法，但它通过"试错"的方式，在发现当前选择无法得到有效解时，回退到上一步重新选择，从而避免了大量无效的计算。

核心思路

  1. 逐行放置：由于每行只能放一个皇后（否则会同行冲突），我们一行一行地处理，每一行只需要选择一个列来放置皇后。
  2. 合法性检查：对于当前行的每一列，检查这个位置是否与已经放置的皇后冲突（列冲突、对角线冲突）。
  3. 递归深入：如果当前位置合法，我们就放置皇后，然后递归处理下一行。
  4. 回溯回退：如果下一行所有列都无法放置皇后，说明当前的选择不对，我们就把当前的皇后移除，尝试下一个列。
  5. 终止条件：当我们成功处理完所有 8 行时，说明我们找到了一个有效的解。

优化：冲突标记

为了快速检查冲突，我们不需要每次都遍历所有已放置的皇后，而是用三个数组来标记状态：

  • col[c]：标记第 c 列是否已经有皇后
  • d1[row-col+7]：标记主对角线（左上到右下）是否有皇后。对于主对角线，row - col 的值是固定的，范围是 -7 到 7，加 7 变成 0 到 14 的索引。
  • d2[row+col]：标记副对角线（右上到左下）是否有皇后。对于副对角线，row + col 的值是固定的，范围是 0 到 14。

✨ 通过这三个标记数组，我们可以在 O(1) 时间内完成冲突检查，将整个算法的效率提升了一个量级！

C++ 完整实现代码

下面是完整的 C++ 回溯法实现代码，包含了上述的优化标记，代码清晰易懂，适合初学者学习。 

#include <iostream>
using namespace std;

const int N = 8;                 // 皇后的数量
bool col[N] = {false};           // 列标记：col[c] 表示第c列是否有皇后
bool d1[2*N] = {false};          // 主对角线标记：row-col
bool d2[2*N] = {false};          // 副对角线标记：row+col
int pos[N] = {0};                // 记录解：pos[row] 表示第row行皇后的列位置
int solution_count = 0;          // 解的计数器

// 打印当前的解
void print_solution() {
    cout << "解 " << ++solution_count << ": " << endl;
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++) {
            if (pos[i] == j) {
                cout << "Q ";
            } else {
                cout << ". ";
            }
        }
        cout << endl;
    }
    cout << endl;
}

// 回溯函数：处理第row行
void backtrack(int row) {
    // 递归终止条件：所有行都处理完了，找到一个解
    if (row == N) {
        print_solution();
        return;
    }

    // 尝试当前行的每一列
    for (int c = 0; c < N; c++) {
        // 检查当前位置是否合法（没有冲突）
        if (!col[c] && !d1[row - c + N - 1] && !d2[row + c]) {
            // 放置皇后，标记状态
            pos[row] = c;
            col[c] = true;
            d1[row - c + N - 1] = true;
            d2[row + c] = true;

            // 递归处理下一行
            backtrack(row + 1);

            // 回溯：撤销当前选择，尝试下一个列
            col[c] = false;
            d1[row - c + N - 1] = false;
            d2[row + c] = false;
        }
    }
}

int main() {
    cout << "八皇后问题的所有解：" << endl << endl;
    backtrack(0);
    cout << "总共有 " << solution_count << " 个解。" << endl;
    return 0;
}

📝 代码说明：你可以将这段代码复制到你的 C++ 编译器中运行，它会输出所有 92 个解的棋盘布局。 如果你只想统计解的数量而不打印每个解，可以注释掉 print_solution() 调用，运行速度会更快。
