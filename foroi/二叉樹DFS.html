C++ 原始碼聯動

C++11 
#include <iostream>#include <vector>using namespace std;vector<int> adj[12]; // 11個節點的多叉樹鄰接表bool visited[12];    // 記錄 1~11 號節點void dfs(int u) {    visited[u] = true; // 1. 標記當前節點已訪問    for (int v : adj[u]) { // 2. 按順序遍歷所有相連鄰居        if (!visited[v]) {            dfs(v);        // 3. 遞歸下探子節點分支        }    }} // 4. 子樹遍歷完畢，自動彈棧並「回溯」int main() {    // 構建多叉四層樹: 1為三叉根；2, 4各有兩個分叉；5, 8各有最底層子葉    adj[1]={2,3,4}; adj[2]={5,6}; adj[3]={7}; adj[4]={8,9};    adj[5]={10};   adj[8]={11};    dfs(1); // 從根節點 1 啟動探索}

多叉四層樹的課堂核心提問

  • 問學生：「當遍歷至 10 (第四層最深點) 時，呼叫棧深度是多少？它是如何從 10 回溯退回至 2 的？」
  • 引導觀察：根節點 1 的三個分叉（左、中、右分支）是如何嚴格按照 DFS 「先往極深處下探、不撞南牆不回頭」的順序逐一掃描完畢的。
