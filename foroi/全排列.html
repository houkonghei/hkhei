核心提示：此演算法的思想在於「固定」當前位置的元素。 對於 index = start 而言，我們嘗試將它與後方所有的 i (包含自己) 進行交換。 交換完後，代表 start 的位置被固定下來了，我們再遞迴處理下一個位置 start+1。 在探索完整個分支返回時，我們必須再執行一次 swap 把位置換回來（這就是回溯法精髓的撤銷選擇），才能保證下一個分支能基於原本乾淨的順序進行。
